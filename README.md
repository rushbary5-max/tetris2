# tetris2
🎮 테트리스 – 고스트 리플레이 대결

이 프로젝트는 현재 플레이 중인 테트리스와 과거 최고 기록 플레이(고스트 리플레이)를 동시에 비교할 수 있는 테트리스 게임이다.

왼쪽에는 이전 최고 기록의 자동 리플레이, 오른쪽에는 현재 플레이 화면이 표시되어 자기 자신과 실시간으로 대결하는 느낌을 제공한다.

🧩 주요 기능 한눈에 보기

고스트 리플레이

최고 점수 플레이를 자동으로 재생

과거 입력(좌/우/회전/드롭)을 시간 기반으로 그대로 재현

현재 플레이

일반 테트리스 조작

점수 실시간 표시

다음 블록 4개 미리보기

최고 기록 저장

최고 점수, 블록 순서, 입력 로그를 localStorage에 저장

새 기록 달성 시 자동 갱신

일시정지 지원

P 키 또는 버튼으로 일시정지

🏗️ 코드 구조 요약 1️⃣ TetrisEngine 클래스 (게임 핵심 엔진) class TetrisEngine

테트리스의 모든 로직을 담당하는 공통 엔진이다.

필드(arena) 관리

블록 생성 및 충돌 판정

회전, 이동, 드롭

줄 제거 및 점수 계산

캔버스 렌더링

👉 현재 플레이와 고스트 리플레이가 같은 엔진을 공유하며 동작한다는 점이 핵심이다.
